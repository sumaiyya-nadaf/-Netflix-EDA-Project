# Netflix Movies & TV Shows — EDA Project

## Overview
Exploratory Data Analysis on the Netflix Titles dataset containing
8807 titles across 12 features.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Google Colab

## Key Findings
- Netflix library is 70% Movies and 30% TV Shows
- USA and India are the top content producing countries
- 2019 was Netflix's peak year with 2097 titles added
- TV-MA is the dominant content rating showing adult focus
- International Movies is the largest genre on the platform

## Project Structure
- `Netflix_EDA_Project.ipynb` — Main analysis notebook
- `chart1_pie.png` through `chart7_heatmap.png` — Visualizations

## How to Run
Open the notebook in Google Colab and run all cells in order.
